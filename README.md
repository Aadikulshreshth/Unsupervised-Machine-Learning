This repository contains my implementations and practice work related to **Unsupervised Machine Learning**, including clustering algorithms, dimensionality reduction techniques, distance measures, and neural network-based representation learning.

---

## Overview

Unsupervised learning is a machine learning approach where models discover hidden patterns, structures, and relationships in **unlabeled data** without predefined target outputs.

This repository serves as a collection of:

* Practice implementations
* Lab experiments
* Data exploration techniques
* Clustering and visualization workflows
* Feature extraction methods
